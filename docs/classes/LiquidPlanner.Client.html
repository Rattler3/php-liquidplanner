clientsCommentsDelete(int $clientId, int $commentId) : string

Deletes a comment on a client from Liquid Planner

Pass the ID of a comment in Liquid Planner into this method and it will be deleted from the workspace. The raw response from the web service is returned so you can examine the result.

Parameters

int $clientId

the ID of the client in Liquid Planner
int $commentId

the ID of the comment in Liquid Planner
